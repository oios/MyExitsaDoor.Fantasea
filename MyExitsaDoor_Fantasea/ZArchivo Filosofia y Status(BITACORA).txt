CUSTOM FOLDER ENCANTACION:
In C++, you can create a folder with a custom icon by using the desktop.ini file. This file allows you to customize the appearance of a folder in Windows. You can specify the icon that you want to use for the folder by setting the IconFile and IconIndex entries in the [.ShellClassInfo] section of the desktop.ini file 1. Here is an example of how to do this:

#include <windows.h>
#include <string>
#include <fstream>

int main() {
    std::string folderPath = "C:\\MyFolder";
    std::string iconPath = "C:\\MyIcon.ico";
    int iconIndex = 0;

    // Create the folder
    CreateDirectory(folderPath.c_str(), NULL);

    // Create the desktop.ini file
    std::ofstream desktopIni((folderPath + "\\desktop.ini").c_str());
    desktopIni << "[.ShellClassInfo]\n";
    desktopIni << "IconResource=" << iconPath << "," << iconIndex << "\n";
    desktopIni.close();

    // Set the folder as a system folder so that the custom icon is displayed
    SetFileAttributes((folderPath + "\\desktop.ini").c_str(), FILE_ATTRIBUTE_HIDDEN | FILE_ATTRIBUTE_SYSTEM);
    SetFileAttributes(folderPath.c_str(), FILE_ATTRIBUTE_READONLY | FILE_ATTRIBUTE_SYSTEM);

    // Notify the shell that the folder icon has changed
    SHChangeNotify(SHCNE_UPDATEITEM, SHCNF_PATH, folderPath.c_str(), NULL);

    return 0;
}
Copiar
This code creates a folder at the specified path, creates a desktop.ini file inside the folder, 
and sets the IconResource entry to specify the path 
and index of the icon that you want to use for the folder. 
It also sets the attributes of the desktop.ini file and the folder itself so that the custom icon is displayed.
 Finally, it notifies the shell that the folder icon has changed so that the new icon is displayed immediately.
